# Scribble

Work on an art-related project for 3 hours - get a sketchbook
1 hour? Get a sticker!

[![My Skills](https://skillicons.dev/icons?i=css,ts,tailwind,react,nextjs)](https://skillicons.dev)

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/jane-does-coding/scribbble
cd scribble
```

### 2. Install the dependencies

```bash
npm i
```

### 2. Run the Website

```bash
npm run dev
```
